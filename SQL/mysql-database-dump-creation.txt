1. login

mysql -u root -p

2. Create a new user and password with the following command:

CREATE USER 'lmlahti' IDENTIFIED BY 'passu';

3. Assign privileges to the new user, enter the command:

GRANT ALL ON *.* TO 'lmlahti';

4. Create Phyloarray database

CREATE DATABASE IF NOT EXISTS Phyloarray;

5. Exit MySQL interface, enter:

exit

6. Load the dump into the database

mysql -ulmlahti -ppassu Phyloarray < /home/lei/data/HITChip/backup_file_hitchip.sql
